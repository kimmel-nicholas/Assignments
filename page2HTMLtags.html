    HTML Tags and Attributes
    Tags Attributes
    Headers - <h1>, <h2>, <h3>,... Similar to the healdlines in a newspaper. Different sizes can be used with 'h1' being the largest then to 'h2' and so on.
    Paragraphs - <p> Similar to the subtext below the headlines in a newspaper which relate to the headline above.
    Images - <img> Images can be sourced from the internet or from the web server by either using a link or a file path.
    Links - <a href> Adding links to a website can make navigating to another part of the site or to another website altogether easier. <a> is used alongside href as the attribute to the hyperlink
    Lists - <ul>, <ol>, <dl> There are many types of lists. The name of each quickly explains each types usage. Unordered List <ul>, Ordered List <ol>, Description List <dl>, and Nested Lists which can use any type of list. Each uses <li> to define elements in the list
